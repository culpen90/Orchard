What is shared

Orchard can request a Google search only while paired. Each search opens in a visible tab. The extension returns up to 8 result titles, links, snippets, and a bounded copy of visible page text to Orchard over this Mac’s loopback connection. Orchard then sends that evidence to the OpenRouter model provider you selected so the model can answer your request.

The token stays in Chrome storage; challenge-response pairing proves both sides without sending the token over the socket. Search tabs remain open for you to inspect. Captcha, consent, and error pages are never returned as search results.

On first connect, Chrome may ask to access devices on your local network. This extension uses that permission only for Orchard at 127.0.0.1 on this Mac.
